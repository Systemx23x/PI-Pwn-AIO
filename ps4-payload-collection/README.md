## Setup PI-PWN Payloads Internal on Raspberry PI
## Add Payloads 11.00
````
rm -r -f /boot/firmware/PPPwn/payloads
cd  ~/PI-PWN/ps4-payload-collection/11.00/
mkdir /boot/firmware/PPPwn/payloads
cp -r *.bin /boot/firmware/PPPwn/payloads
````
